# E-commerce Cart App (React + Redux Toolkit)

A simple e-commerce cart application built with **React** and **Redux Toolkit**. This project demonstrates product listing, cart management, and a responsive UI with interactive features.

---

## Features

- Product listing with images, price, and rating.
- Add to Cart functionality with **Redux Toolkit** state management.
- Increment / Decrement product quantity.
- Manual quantity input for direct updates.
- Empty Cart functionality.
- Cart totals calculation (price and quantity).
- Conditional rendering: empty cart message and cart summary header.
- Toaster notifications for actions like add/remove items.
- Responsive layout with Bootstrap and custom CSS.
- Scroll to top on cart actions.

---

## Technologies Used

- React
- Redux Toolkit
- React Hot Toast
- Bootstrap
- Vite (for build and development)
- CSS / Custom styling
